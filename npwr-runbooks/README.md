## Files

- `company.md` — Who NorthernPWR is, stats, locations, contact
- `solar-101.md` — How solar works, components, terminology
- `install-process.md` — The NorthernPWR install journey start to finish
- `faq.md` — Common homeowner questions and answers
- `support-runbook.md` — Procedures for the support agent
- `sales-runbook.md` — Procedures for the sales agent
